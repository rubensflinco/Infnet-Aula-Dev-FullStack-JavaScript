# Infnet-Aula-Dev-FullStack-JavaScript

## Depencias:
- NodeJS
- NPM

## Rodando Front:
- Acesse a pasta com o comando ```cd front```.
- Para iniciar use o comando ```npm run dev```.
- Pronto.

## Rodando Back:
- Acesse a pasta com o comando ```cd back```.
- Para iniciar use o comando ```npm run dev```.
- Pronto.
